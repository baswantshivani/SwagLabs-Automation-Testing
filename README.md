# SwagLabs E-Commerce — Automation Testing

## Project Overview
Automated testing of the SwagLabs e-commerce web application 
using Selenium WebDriver with Java.

## Tech Stack
- Selenium WebDriver
- Java
- TestNG
- Page Object Model (POM)
- Data-Driven Testing (Apache POI)

## Modules Tested
- Login (Positive & Negative scenarios)
- Product Listing & Details
- Add to Cart
- Checkout Flow

## How to Run
1. Clone the repository
2. Open in Eclipse or IntelliJ IDEA
3. Run individual test files or testng.xml
4. View HTML reports generated by TestNG
